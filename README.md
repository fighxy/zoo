# Zoo Store

A Next.js e-commerce application powered by PocketBase.

## Quick Start

### 1. Start Backend
```bash
cd backend
./pocketbase.exe serve
```
Access Admin UI at http://127.0.0.1:8090/_/
Credentials: `admin@zoo.local` / `1234567890`

### 2. Start Frontend
```bash
npm run dev
```
Access site at http://localhost:3000

## Features
- **Next.js 15** (App Router)
- **PocketBase** (Database, Auth, Files, Admin UI)
- **TailwindCSS**
- **TypeScript**

## Data Management
Manage products, categories, and images via the PocketBase Admin UI.
